WifiProxySwitcher 使用手册


# 应用简介

WifiProxySwitcher 是一款便捷的 WiFi 网络和代理设置切换工具，允许用户为不同的 WiFi 网络预设代理配置，并通过全局快捷键快速切换。

# 主要功能

• 📶 一键切换 WiFi 网络

• 🔄 自动设置/取消代理

• ⌨️ 全局热键支持

• 📊 多配置管理

• 🎯 系统托盘运行

# 系统要求

最低配置

• 操作系统: Windows 10 或更高版本

• .NET 框架: .NET 6.0 或更高版本

• 内存: 至少 512MB RAM

• 权限: 管理员权限（用于修改系统代理设置）

# 推荐配置

• 操作系统: Windows 11

• 内存: 1GB RAM 或更高

• 磁盘空间: 至少 50MB 可用空间

# 安装说明

1. 下载 WifiProxySwitcher.exe 可执行文件
2. 双击 WifiProxySwitcher.exe 运行即可使用
3. 配置文件将自动创建在程序同目录下


# 区域 功能说明

快捷键列 显示/设置全局热键，点击可重新捕获

WiFi名称 目标 WiFi 网络的 SSID 名称

是否开启代理 复选框，控制是否启用代理

代理服务器地址 代理服务器地址和端口（如：127.0.0.1:8080）

应用按钮 立即应用该行的配置

保存按钮 保存所有配置更改

调试窗口 显示操作日志和状态信息

# 使用教程

## 第一步：添加新配置

1. 在表格空白行点击相应单元格
2. 设置快捷键：点击"快捷键"列单元格，按下想要的组合键（如 Ctrl+Alt+F1）
3. 输入目标 WiFi 名称
4. 勾选"是否开启代理"（如需要）
5. 输入代理服务器地址（格式：IP:端口）

第二步：设置全局热键

1. 点击快捷键列的单元格
2. 按下想要设置的组合键，支持：

   • 修饰键: Ctrl、Shift、Alt

   • 功能键: F1-F12

3. 系统会自动捕获并显示热键组合
4. 注意: 必须包含至少一个非修饰键

第三步：保存配置

1. 完成所有配置后，点击"保存"按钮
2. 系统会提示保存成功
3. 配置自动生效，热键立即可用

第四步：使用热键切换

• 在任何应用程序中按下设置的全局热键

• 系统会自动：

  1. 切换到指定 WiFi 网络
  2. 启用/禁用代理设置
  3. 在系统托盘显示操作结果

# 常见问题

## Q:一次切换不成功，需要多次切换？

A ： win系统问题，可能需要热键两次才能生效

## Q: 热键无法正常使用？

A: 检查以下可能原因：
• 热键已被其他程序占用

• 缺少非修饰键（必须包含字母/数字/功能键）

• 程序未以管理员权限运行

## Q: WiFi 切换失败？

A: 确保：
• 目标 WiFi 网络在可连接范围内

• WiFi 名称输入正确（区分大小写）

• 系统 WiFi 功能正常

## Q: 代理设置不生效？

A: 验证：
• 代理服务器地址格式正确（IP:端口）

• 代理服务器可正常连接

• 系统代理设置未被其他程序修改

## Q: 程序无法启动？

A: 尝试：
• 安装 .NET 6.0 或更高版本运行时

• 以管理员身份运行

• 检查杀毒软件是否误拦截

# 故障排除

基本检查步骤

1. 查看调试窗口: 主界面下方的调试窗口会显示详细错误信息
2. 检查配置文件: 程序目录下的 config.json 文件可手动编辑
3. 重启程序: 简单的重启可能解决临时问题

配置文件示例

[
  {
    "ShortcutKey": "Ctrl+Alt+F1",
    "WifiName": "MyHomeWiFi",
    "EnableProxy": true,
    "ProxyAddress": "127.0.0.1:8080",
    "HotkeyId": 12345
  }
]


手动编辑配置

如果图形界面出现问题，可以：
1. 关闭程序
2. 用文本编辑器打开 config.json
3. 按 JSON 格式修改配置
4. 重新启动程序

重置配置

删除 config.json 文件，程序会在下次启动时创建新的默认配置。

提示: 使用过程中遇到问题，请先查看调试输出窗口的详细日志信息，这些信息有助于快速定位问题原因。
